# dben-releases

Public distribution channel for [DBen](https://github.com/starkross/dben)
(source is private). This repo hosts:

- **DMGs** — as GitHub Release assets.
- **`appcast.xml`** — the Sparkle update feed, served via GitHub Pages at
  `https://starkross.github.io/dben-releases/appcast.xml`.

Both are published automatically by the `release.yml` workflow in the private
`dben` repo. Nothing here is edited by hand.
